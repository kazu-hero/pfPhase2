このディレクトリには、タスク依存関係グラフを表示する2つのサンプルHTMLファイルがあります：

## 📌 Cytoscape.js とは？

**グラフ理論に特化したネットワーク可視化ライブラリ**

### 特徴
- ✅ **グラフ/ネットワーク専用**に設計されている
- ✅ **レイアウトアルゴリズムが豊富**（階層型、放射状、力学シミュレーションなど）
- ✅ **学習コストが低い**（グラフ専用なので、設定がシンプル）
- ✅ **ノードやエッジの操作がしやすい**
- ✅ **モバイル対応**のタッチイベントもサポート

### 向いている用途
- ✅ ネットワーク図、組織図、依存関係グラフ
- ✅ ソーシャルネットワーク分析
- ✅ **今回のようなタスク依存関係の可視化に最適**

### コード例（シンプル）
```javascript
var cy = cytoscape({
    container: document.getElementById('cy'),
    elements: [
        { data: { id: 'a', label: 'ノードA' } },
        { data: { id: 'b', label: 'ノードB' } },
        { data: { source: 'a', target: 'b' } }
    ],
    style: [
        {
            selector: 'node',
            style: {
                'background-color': '#6ab04c',
                'label': 'data(label)'
            }
        }
    ],
    layout: { name: 'breadthfirst' }
});
```

## 📌 D3.js とは？

**データ駆動型のドキュメント操作ライブラリ**

### 特徴
- ✅ **超カスタマイズ可能**（SVG要素を完全にコントロール）
- ✅ **グラフ以外にも使える**（棒グラフ、円グラフ、地図など）
- ✅ **アニメーションが豊富**
- ✅ **データバインディング**が強力
- ⚠️ **学習コストが高い**（低レベルのSVG操作が必要）
- ⚠️ **コード量が多くなりがち**

### 向いている用途
- ✅ 複雑なカスタムビジュアライゼーション
- ✅ データダッシュボード
- ✅ 統計グラフ、チャート全般
- ⚠️ シンプルなネットワーク図には過剰

### コード例（やや複雑）
```javascript
const nodes = [
    { id: 'a', label: 'ノードA' },
    { id: 'b', label: 'ノードB' }
];
const links = [
    { source: 'a', target: 'b' }
];

const simulation = d3.forceSimulation(nodes)
    .force('link', d3.forceLink(links).id(d => d.id))
    .force('charge', d3.forceManyBody())
    .force('center', d3.forceCenter(width / 2, height / 2));

const link = svg.selectAll('line')
    .data(links)
    .enter().append('line');

const node = svg.selectAll('circle')
    .data(nodes)
    .enter().append('circle')
    .attr('r', 10);

simulation.on('tick', () => {
    link.attr('x1', d => d.source.x)
        .attr('y1', d => d.source.y)
        .attr('x2', d => d.target.x)
        .attr('y2', d => d.target.y);
    
    node.attr('cx', d => d.x)
        .attr('cy', d => d.y);
});
```

## 💡 今回のプロジェクトへの推奨

### **Cytoscape.js を推奨します** ⭐

#### 理由
1. ✅ **タスク依存関係グラフに最適**
2. ✅ **学習コストが低く、開発スピードが速い**
3. ✅ **レイアウトアルゴリズムが豊富**（階層型、ツリー型など）
4. ✅ **Salesforce LWCとの統合が簡単**
5. ✅ **モバイル対応が標準**
6. ✅ **コード量が少なく、メンテナンスしやすい**

#### D3.jsを選ぶべきケース
- 非常に複雑なカスタムビジュアライゼーションが必要
- グラフ以外の要素（チャート、タイムラインなど）も同時に表示
- SVGレベルでの細かいコントロールが必須

## ❓ よくある質問

### Q: Salesforce LWCで使えますか？
**A:** はい、両方とも使えます。静的リソースとしてライブラリをアップロードして、LWCで読み込むことができます。

### Q: どちらが軽量ですか？
**A:** ファイルサイズはCytoscape.jsの方がやや小さいです（約500KB vs 約250KB）。ただし、実際のパフォーマンスはほぼ同等です。

### Q: 既存のSalesforceデータと連携できますか？
**A:** はい、Apexで取得したデータをJSON形式でLWCに渡せば、どちらのライブラリでも表示できます。
