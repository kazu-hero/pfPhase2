# 画面構成とUI設計

## **このファイルの目的**

Lightning App Pageの構成、レイアウト、SLDSスタイルを定義します。

## **📐 全体レイアウト**

### **3カラムレイアウト**

```
┌─────────────────────────────────────────────────────┐
│              開発プロセス可視化                      │
├──────────┬────────────────────┬────────────────────┤
│          │                    │                    │
│ フィルター │   グラフ表示        │   詳細パネル        │
│ パネル    │                    │                    │
│          │                    │                    │
│ 25%      │      50%           │      25%           │
│          │                    │                    │
└──────────┴────────────────────┴────────────────────┘
```

### **画面サイズ別の対応**

| 画面サイズ | フィルター | グラフ | 詳細 |
|-----------|----------|-------|------|
| デスクトップ | 25% | 50% | 25% |
| タブレット | 100% | 100% | 100% (縦並び) |
| モバイル | 100% | 100% | 100% (縦並び) |

## **🎨 Lightning Design System（SLDS）の使用**

### **1. カードコンポーネント**

```html
<lightning-card title="タイトル">
    <div slot="actions">
        <!-- アクションボタン -->
    </div>
    <div class="slds-p-around_small">
        <!-- コンテンツ -->
    </div>
</lightning-card>
```

**使用するSLDSクラス**:
- `slds-card` - カード基本
- `slds-p-around_small` - 内側余白（小）
- `slds-m-bottom_small` - 下側マージン（小）

---

### **2. グリッドレイアウト**

```html
<div class="slds-grid slds-wrap">
    <div class="slds-col slds-size_1-of-4">
        <!-- 左カラム（25%） -->
    </div>
    <div class="slds-col slds-size_1-of-2">
        <!-- 中央カラム（50%） -->
    </div>
    <div class="slds-col slds-size_1-of-4">
        <!-- 右カラム（25%） -->
    </div>
</div>
```

**レスポンシブ対応**:
```html
<div class="slds-col slds-size_1-of-1 slds-medium-size_1-of-4">
    <!-- モバイル: 100%, タブレット以上: 25% -->
</div>
```

---

### **3. ボタン**

```html
<!-- プライマリボタン -->
<lightning-button
    label="適用"
    variant="brand"
    onclick={handleApply}>
</lightning-button>

<!-- セカンダリボタン -->
<lightning-button
    label="クリア"
    variant="neutral"
    onclick={handleClear}>
</lightning-button>

<!-- アイコンボタン -->
<lightning-button-icon
    icon-name="utility:zoomin"
    alternative-text="ズームイン"
    onclick={handleZoomIn}>
</lightning-button-icon>
```

**使用するバリアント**:
- `brand` - プライマリアクション（青）
- `neutral` - セカンダリアクション（グレー）
- `destructive` - 削除系（赤）

---

### **4. チェックボックスグループ**

```html
<lightning-checkbox-group
    name="statusFilter"
    label="ステータス"
    options={statusOptions}
    value={selectedStatuses}
    onchange={handleStatusChange}>
</lightning-checkbox-group>
```

**オプション形式**:
```javascript
statusOptions = [
    { label: '未着手 (TODO)', value: 'TODO' },
    { label: '作業中 (WIP)', value: 'WIP' },
    { label: '完了 (DONE)', value: 'DONE' },
    { label: '対応中 (OPEN)', value: 'OPEN' },
    { label: 'クローズ (CLOSE)', value: 'CLOSE' }
];
```

---

### **5. ラジオグループ**

```html
<lightning-radio-group
    name="originObjectType"
    label="起点オブジェクト"
    options={originObjectOptions}
    value={originObjectType}
    onchange={handleOriginObjectChange}>
</lightning-radio-group>
```

---

### **6. バッジ**

```html
<lightning-badge label={status}></lightning-badge>
```

**ステータス別の色**:
```css
/* カスタムCSSで色をオーバーライド */
.status-todo {
    background-color: #999999;
}
.status-wip {
    background-color: #FFC107;
}
.status-done {
    background-color: #4CAF50;
}
```

---

### **7. アイコン**

```html
<lightning-icon
    icon-name="utility:hide"
    size="x-small"
    alternative-text="非表示">
</lightning-icon>
```

**使用するアイコン**:
- `utility:hide` - フィルター外レコード
- `utility:zoomin` - ズームイン
- `utility:zoomout` - ズームアウト
- `utility:expand_alt` - フィット
- `utility:refresh` - リセット

## **📱 レスポンシブデザイン**

### **ブレークポイント**

| サイズ | 幅 | 対応 |
|-------|---|------|
| モバイル | < 768px | 縦並び |
| タブレット | 768px - 1024px | 2カラム |
| デスクトップ | > 1024px | 3カラム |

### **メディアクエリ**

```css
/* モバイル（デフォルト） */
.slds-col {
    width: 100%;
}

/* タブレット */
@media (min-width: 768px) {
    .slds-medium-size_1-of-2 {
        width: 50%;
    }
    .slds-medium-size_1-of-4 {
        width: 25%;
    }
}

/* デスクトップ */
@media (min-width: 1024px) {
    .slds-large-size_1-of-4 {
        width: 25%;
    }
    .slds-large-size_1-of-2 {
        width: 50%;
    }
}
```

## **🖼️ Lightning App Pageの設定**

### **ページ作成手順**

1. **Setup → Lightning App Builder → New**
2. **App Page を選択**
3. **Three Regions を選択**
4. **ページ名**: 「開発プロセス可視化」

### **コンポーネント配置**

| 領域 | コンポーネント | 幅 |
|-----|--------------|---|
| 左 | relationGraphApp (フィルターパネル) | 3 |
| 中央 | relationGraphApp (グラフ表示) | 6 |
| 右 | relationGraphApp (詳細パネル) | 3 |

**注意**: `relationGraphApp` は1つだけ配置し、内部で3つのパネルを表示します。

## **📏 スペーシング**

### **余白の基準**

| サイズ | クラス | ピクセル |
|-------|-------|---------|
| なし | none | 0px |
| 極小 | xxx-small | 2px |
| 小 | small | 12px |
| 中 | medium | 24px |
| 大 | large | 36px |

### **使用例**

```html
<div class="slds-p-around_small">
    <!-- 内側余白: 12px -->
</div>

<div class="slds-m-bottom_medium">
    <!-- 下側マージン: 24px -->
</div>
```
