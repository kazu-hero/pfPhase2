# 詳細パネル仕様

## **概要**

ノードクリック時に右側パネルに表示される詳細情報の仕様です。
レコードページへのハイパーリンクと、フィルター外の関連レコード表示が特徴です。

## **2. レコードページへのハイパーリンク**

### **URL生成ロジック**

```javascript
// レコードURLの生成
get recordUrl() {
    const baseUrl = window.location.origin;
    return `${baseUrl}/lightning/r/${this.objectType}/${this.recordId}/view`;
}

// 関連レコードのURL生成
buildRecordUrl(objectType, recordId) {
    const baseUrl = window.location.origin;
    return `${baseUrl}/lightning/r/${objectType}/${recordId}/view`;
}
```

### **リンクの動作**

```html
<!-- 新しいタブで開く -->
<a href={recordUrl} target="_blank" rel="noopener noreferrer">
    {recordName}
</a>

<!-- 同じタブで開く（オプション） -->
<a href={recordUrl} onclick={handleNavigate}>
    {recordName}
</a>
```

### **NavigationMixinを使用する場合**

```javascript
import { NavigationMixin } from 'lightning/navigation';

export default class DetailPanel extends NavigationMixin(LightningElement) {
    
    handleNavigate(event) {
        event.preventDefault();
        
        this[NavigationMixin.Navigate]({
            type: 'standard__recordPage',
            attributes: {
                recordId: this.recordId,
                objectApiName: this.objectType,
                actionName: 'view'
            }
        });
    }
}
```

## **3. フィルター外関連レコードの表示**

### **データ取得**

```javascript
// ノードクリック時の処理
async handleNodeClick(nodeId, objectType) {
    try {
        // 全関連レコードを取得（フィルター無視）
        const relatedData = await getRelatedRecords({
            recordId: nodeId,
            objectType: objectType
        });
        
        // 現在のフィルター条件と照合
        this.upstreamRecords = this.markFilteredRecords(relatedData.upstreamRecords);
        this.downstreamRecords = this.markFilteredRecords(relatedData.downstreamRecords);
        
    } catch (error) {
        this.showToast('エラー', '関連レコードの取得に失敗しました', 'error');
    }
}
```

### **フィルター外判定ロジック**

```javascript
markFilteredRecords(records) {
    return records.map(record => {
        // 現在のフィルター条件に合致するか判定
        const isStatusMatch = this.selectedStatuses.includes(record.status);
        const isTypeMatch = this.selectedObjectTypes.includes(record.objectType);
        const isFilteredOut = !(isStatusMatch && isTypeMatch);
        
        return {
            ...record,
            isFilteredOut: isFilteredOut,
            cssClass: isFilteredOut ? 'record-item filtered-out' : 'record-item',
            url: this.buildRecordUrl(record.objectType, record.id)
        };
    });
}
```

## **4. Apex側の実装**

### **関連レコード取得メソッド**

```apex
@AuraEnabled
public static RelatedRecordsData getRelatedRecords(
    Id recordId,
    String objectType
) {
    RelatedRecordsData result = new RelatedRecordsData();
    
    // オブジェクトタイプに応じて処理を分岐
    switch on objectType {
        when 'Dev_Service__c' {
            // 上流なし
            result.upstreamRecords = new List<RecordInfo>();
            // 下流取得
            result.downstreamRecords = fetchDownstreamFromService(recordId);
        }
        when 'Issue__c' {
            // 上流取得
            result.upstreamRecords = fetchUpstreamFromIssue(recordId);
            // 下流取得
            result.downstreamRecords = fetchDownstreamFromIssue(recordId);
        }
        when 'TechActionPlan__c' {
            result.upstreamRecords = fetchUpstreamFromPlan(recordId);
            result.downstreamRecords = fetchDownstreamFromPlan(recordId);
        }
        when 'FunctionRequirements__c' {
            result.upstreamRecords = fetchUpstreamFromRequirements(recordId);
            result.downstreamRecords = fetchDownstreamFromRequirements(recordId);
        }
        when 'Dev_Ticket__c' {
            // 上流取得
            result.upstreamRecords = fetchUpstreamFromTicket(recordId);
            // 下流なし
            result.downstreamRecords = new List<RecordInfo>();
        }
    }
    
    return result;
}
```

### **データ構造**

```apex
public class RelatedRecordsData {
    @AuraEnabled public List<RecordInfo> upstreamRecords { get; set; }
    @AuraEnabled public List<RecordInfo> downstreamRecords { get; set; }
}

public class RecordInfo {
    @AuraEnabled public Id id { get; set; }
    @AuraEnabled public String name { get; set; }
    @AuraEnabled public String objectType { get; set; }
    @AuraEnabled public String status { get; set; }
    @AuraEnabled public String ownerName { get; set; }
    @AuraEnabled public Date dueDate { get; set; }
}
```

### **下流取得の例（Issue → TechActionPlan）**

```apex
private static List<RecordInfo> fetchDownstreamFromIssue(Id issueId) {
    List<RecordInfo> records = new List<RecordInfo>();
    
    // 中間オブジェクトを経由してTechActionPlanを取得
    List<Relation_Issue2Plan__c> relations = [
        SELECT TechActionPlan__c, 
               TechActionPlan__r.Name,
               TechActionPlan__r.Status__c,
               TechActionPlan__r.Owner.Name,
               TechActionPlan__r.Due_Date__c
        FROM Relation_Issue2Plan__c
        WHERE Issue__c = :issueId
    ];
    
    for (Relation_Issue2Plan__c rel : relations) {
        RecordInfo info = new RecordInfo();
        info.id = rel.TechActionPlan__c;
        info.name = rel.TechActionPlan__r.Name;
        info.objectType = 'TechActionPlan__c';
        info.status = rel.TechActionPlan__r.Status__c;
        info.ownerName = rel.TechActionPlan__r.Owner.Name;
        info.dueDate = rel.TechActionPlan__r.Due_Date__c;
        records.add(info);
    }
    
    // さらに下流（TechActionPlan → FunctionRequirements）も取得
    for (RecordInfo plan : records) {
        records.addAll(fetchDownstreamFromPlan(plan.id));
    }
    
    return records;
}
```

## **5. ステータスバッジのスタイル**

### **バッジ色の設定**

```javascript
get statusClass() {
    switch(this.status) {
        case 'TODO':
            return 'slds-badge slds-theme_inverse';
        case 'WIP':
            return 'slds-badge slds-theme_warning';
        case 'DONE':
            return 'slds-badge slds-theme_success';
        case 'OPEN':
            return 'slds-badge slds-theme_warning';
        case 'CLOSE':
            return 'slds-badge slds-theme_info';
        default:
            return 'slds-badge';
    }
}
```

## **7. エラーハンドリング**

```javascript
async loadRecordDetails(recordId, objectType) {
    try {
        const data = await getRelatedRecords({ recordId, objectType });
        
        if (!data || (!data.upstreamRecords && !data.downstreamRecords)) {
            this.showToast('警告', '関連レコードが見つかりませんでした', 'warning');
            return;
        }
        
        this.processRecordData(data);
        
    } catch (error) {
        console.error('Record details load error:', error);
        this.showToast('エラー', '詳細情報の読み込みに失敗しました', 'error');
    }
}
```
