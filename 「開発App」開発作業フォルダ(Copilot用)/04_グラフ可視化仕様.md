# グラフ可視化仕様

## **概要**

Cytoscape.jsを使用して5階層のオブジェクト関係を視覚的に表現します。

---

## **Cytoscape.js 設定**

### **基本設定**

```javascript
const cy = cytoscape({
    container: this.template.querySelector('.graph-container'),
    
    // レイアウト設定
    layout: {
        name: 'dagre', // または 'breadthfirst'
        rankDir: 'LR', // Left to Right (左→右)
        spacingFactor: 1.5, // ノード間隔
        nodeDimensionsIncludeLabels: true
    },
    
    // スタイル設定
    style: [
        // ノードの基本スタイル
        {
            selector: 'node',
            style: {
                'label': 'data(label)',
                'text-valign': 'center',
                'text-halign': 'center',
                'font-size': '12px',
                'width': 60,
                'height': 60
            }
        },
        // エッジの基本スタイル
        {
            selector: 'edge',
            style: {
                'width': 2,
                'line-color': '#ccc',
                'target-arrow-color': '#ccc',
                'target-arrow-shape': 'triangle',
                'curve-style': 'bezier'
            }
        }
    ],
    
    // インタラクション設定
    minZoom: 0.5,
    maxZoom: 2,
    wheelSensitivity: 0.2
});
```

## **ノードのビジュアル設計**

### **オブジェクトタイプ別の形状**

| オブジェクト | 形状 | Cytoscape.js設定 | 意味 |
|-------------|------|-----------------|------|
| Dev_Service__c | 六角形 | `shape: 'hexagon'` | サービスの起点を強調 |
| Issue__c | 円形 | `shape: 'ellipse'` | 課題の柔軟性 |
| TechActionPlan__c | 四角形 | `shape: 'rectangle'` | 施策の構造性 |
| FunctionRequirements__c | 菱形 | `shape: 'diamond'` | 要件の重要性 |
| Dev_Ticket__c | 楕円形 | `shape: 'ellipse'` | タスクの実行 |

### **ステータス別の色**

**⚠️ 注意**: Dev_Service__c は独自のステータス値（OPEN/CLOSE）を使用します。

**開発チケット系（Issue, TechActionPlan, FunctionRequirements, Dev_Ticket）**:
| ステータス | 色 | カラーコード | 用途 |
|-----------|---|-------------|------|
| TODO | グレー | `#999999` | 未着手タスク |
| WIP | 黄色 | `#FFC107` | 進行中タスク |
| DONE | 緑 | `#4CAF50` | 完了タスク |

**問い合わせ系（Dev_Service）**:
| ステータス | 色 | カラーコード | 用途 |
|-----------|---|-------------|------|
| OPEN | オレンジ | `#FF9800` | 対応中 |
| CLOSE | 青 | `#2196F3` | クローズ済み |

## **レイアウトアルゴリズム**

### **推奨: Dagreレイアウト**

```javascript
{
    name: 'dagre',
    rankDir: 'LR', // 左から右へ
    nodeSep: 50,   // ノードの横間隔
    rankSep: 100,  // 階層間の縦間隔
    align: 'UL',   // 上左揃え
    ranker: 'network-simplex'
}
```

**メリット:**
- 階層構造が明確
- 多対多の関係も表現可能
- パフォーマンスが良い

### **代替: Breadthfirstレイアウト**

```javascript
{
    name: 'breadthfirst',
    directed: true,
    spacingFactor: 1.75,
    avoidOverlap: true,
    maximal: false
}
```

**メリット:**
- シンプルなツリー構造に適している
- レイアウト計算が高速

## **インタラクション実装**

### **ノードクリックイベント**

```javascript
cy.on('tap', 'node', (event) => {
    const node = event.target;
    const nodeData = {
        id: node.data('id'),
        type: node.data('type'),
        name: node.data('label'),
        status: node.data('status')
    };
    
    // LWCイベント発火
    this.dispatchEvent(new CustomEvent('nodeclick', {
        detail: nodeData
    }));
});
```

### **ホバー時のツールチップ**

```javascript
cy.on('mouseover', 'node', (event) => {
    const node = event.target;
    node.style({
        'border-width': 5,
        'border-color': '#e67e22'
    });
    
    // ツールチップ表示
    this.showTooltip(node.data());
});

cy.on('mouseout', 'node', (event) => {
    const node = event.target;
    node.style({
        'border-width': 2
    });
    
    // ツールチップ非表示
    this.hideTooltip();
});
```

### **パスハイライト**

```javascript
highlightPath(nodeId) {
    // 全ノード/エッジを淡色化
    cy.elements().style({
        'opacity': 0.2
    });
    
    // 選択ノード
    const selectedNode = cy.getElementById(nodeId);
    
    // 上流と下流のノードを取得
    const predecessors = selectedNode.predecessors();
    const successors = selectedNode.successors();
    
    // 関連要素を強調
    selectedNode.union(predecessors).union(successors).style({
        'opacity': 1
    });
}
```

## **ズーム・パン操作**

### **ズームコントロール**

```javascript
// ズームイン
zoomIn() {
    cy.zoom({
        level: cy.zoom() * 1.2,
        renderedPosition: { x: cy.width() / 2, y: cy.height() / 2 }
    });
}

// ズームアウト
zoomOut() {
    cy.zoom({
        level: cy.zoom() / 1.2,
        renderedPosition: { x: cy.width() / 2, y: cy.height() / 2 }
    });
}

// フィット（全体表示）
fit() {
    cy.fit();
}

// リセット
reset() {
    cy.reset();
}
```

### **Cytoscape.jsへの変換**

```javascript
renderGraph(graphData) {
    // 既存のグラフをクリア
    cy.elements().remove();
    
    // ステータス色のマッピング（オブジェクトタイプごとに異なるロジック）
    const processedData = {
        nodes: graphData.nodes.map(node => {
            const statusColor = this.getStatusColor(node.data.status, node.data.type);
            return {
                ...node,
                data: {
                    ...node.data,
                    statusColor: statusColor
                }
            };
        }),
        edges: graphData.edges
    };
    
    // ノードとエッジを追加
    cy.add(processedData.nodes);
    cy.add(processedData.edges);
    
    // レイアウト適用
    cy.layout(this.layoutConfig).run();
}

// ステータスに応じた色を返すヘルパーメソッド
getStatusColor(status, objectType) {
    // Dev_Service__c は独自のステータス
    if (objectType === 'Dev_Service__c') {
        switch(status) {
            case 'OPEN': return '#FF9800';  // オレンジ
            case 'CLOSE': return '#2196F3'; // 青
            default: return '#999999';
        }
    }
    
    // その他のオブジェクト（Issue, TechActionPlan, FunctionRequirements, Dev_Ticket）
    switch(status) {
        case 'TODO': return '#999999';  // グレー
        case 'WIP': return '#FFC107';   // 黄色
        case 'DONE': return '#4CAF50';  // 緑
        default: return '#999999';
    }
}
```

## **パフォーマンス最適化**

### **大量ノード時の対策**

```javascript
// ノード数制限
const MAX_VISIBLE_NODES = 500;

if (graphData.nodes.length > MAX_VISIBLE_NODES) {
    this.showWarning('表示するノード数が多すぎます。フィルターを使用してください。');
    return;
}

// レイアウト計算の非同期化
cy.layout({
    name: 'dagre',
    animate: false, // アニメーション無効化で高速化
    stop: () => {
        console.log('Layout complete');
    }
}).run();
```
