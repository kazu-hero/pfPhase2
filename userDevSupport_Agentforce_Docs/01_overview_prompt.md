# 01 Overview Prompt

## 目的
- AIが作業開始前にプロジェクトの全体感を短時間で把握するための概要。

## 最小確認項目（全体把握に必要なものだけ）
- プロジェクト目的: [この開発で達成したいこと]
- 対象範囲: [今回触る機能/領域]
- 非対象範囲: [今回触らない機能/領域]
- 成果物: [作るもの・更新するもの]
- 主な制約: [技術/運用/組織ルール]
- 依存関係: [先に確定が必要な仕様や外部要素]
- 完了条件: [受け入れ判断の条件]

## 利用ルール
- 詳細仕様は `02_solution_definition.md` と `05_requirement_definition.md` を参照する。
- 不明点は推測せず、質問して確定する。
